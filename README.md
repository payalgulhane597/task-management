# Task Management Dashboard

A simple Task Management Dashboard built using Next.js, TypeScript, and Tailwind CSS.

## Features

- Dashboard Overview
- Total Tasks Count
- Completed Tasks Count
- Pending Tasks Count
- Team Members Count
- Responsive UI
- Reusable Components

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Project Setup

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd task-management-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open in browser

```bash
http://localhost:3000
```

## Build for Production

```bash
npm run build
npm start
```
